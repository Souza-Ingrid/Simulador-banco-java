# 💳 Simulador de Conta Bancária

## 📌 Descrição
Este projeto foi desenvolvido como parte de um desafio prático da plataforma DIO (Digital Innovation One), durante o curso de Formação Java.

A aplicação simula a criação de uma conta bancária via terminal, permitindo que o usuário insira seus dados e receba uma mensagem personalizada ao final do processo.

---

## 🎯 Objetivo
Praticar conceitos fundamentais da linguagem Java, incluindo:

- Entrada de dados com `Scanner`
- Tipos de variáveis (`int`, `String`, `double`)
- Estrutura básica de um programa Java
- Concatenação de Strings
- Interação com o usuário via terminal

---

## ⚙️ Funcionalidades
- Solicita ao usuário:
  - Número da conta
  - Agência
  - Nome completo
  - Saldo
- Armazena os dados informados
- Exibe uma mensagem final personalizada com os dados inseridos

---

## 🛠️ Tecnologias utilizadas
- Java
- IntelliJ IDEA
- Git e GitHub

---

## ▶️ Como executar o projeto

1. Clone este repositório:
```bash
git clone https://github.com/Souza-Ingrid/Simulador-banco-java.git
```

2. Abra o projeto em uma IDE (IntelliJ, Eclipse ou VS Code)
3. Execute a classe:
  `ContaTerminal.java`
4. Insira os dados solicitados no terminal

## 🧠 Aprendizados

Durante o desenvolvimento deste projeto, foram praticados conceitos importantes como:

- Manipulação de entrada de dados no Java
- Diferença entre next(), nextLine(), nextInt() e nextDouble()
- Tratamento de buffer do Scanner
- Organização de código
- Uso de Git via terminal para versionamento
